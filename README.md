# 일간 마케팅 대시보드

광고 소재별 ROAS·회원수 성과를 매일 아침 빠르게 파악하기 위한 Streamlit 대시보드입니다.

## 실행 방법

```bash
# 1. 의존성 설치 (최초 1회)
pip install -r requirements.txt

# 2. 앱 실행
streamlit run app.py
```

브라우저에서 자동으로 `http://localhost:8501` 이 열립니다.

---

## 데이터 입력 방법

사이드바에서 세 가지 방법 중 하나를 선택합니다.

### 1) 샘플 데이터 사용
`sample_data/` 폴더의 예제 데이터로 바로 테스트합니다.

### 2) 파일 업로드
- **입력1 (회원 현황)** — `members.csv` 또는 Excel
  컬럼 순서: `날짜, 자사몰 회원수, 네이버 관심고객 증감율(%), 자사몰 당일 매출`

- **입력2 (네이버 광고)** — `naver_ads.csv` 또는 Excel
  컬럼 순서: `날짜, 상품명, 구매전환수, 전환당단가, 총전환비용, 총비용, CPC, 총구매전환매출, CTR(%)`
  > ROAS는 앱이 자동 계산합니다 (총구매전환매출 ÷ 총비용 × 100).

### 3) 직접 붙여넣기
Excel에서 범위를 복사(Ctrl+C) 후 텍스트 박스에 붙여넣기(Ctrl+V).
탭·쉼표 구분 모두 지원합니다.

## 화면 구성

| 영역 | 내용 |
|------|------|
| **표 1** | 선택 날짜의 자사몰 회원수, 전일 대비 증감(자동 계산), 네이버 관심고객 증감율, 당일 매출 |
| **표 2** | 선택 상품의 자사몰 캠페인 — 구매전환수, 전환당단가, 총전환비용, ROAS |
| **표 3** | 선택 상품의 네이버스토어 — CPC, CTR, 총비용, 총구매전환매출, ROAS |

- 증감 수치: 양수=초록, 음수=빨강으로 강조
- 상품 드롭다운 변경 시 표 2·3 즉시 갱신
